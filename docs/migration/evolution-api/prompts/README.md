## Como usar

1. Abrir nueva sesion de Claude Code (contexto limpio).
2. Copiar el contenido del archivo de la sesion correspondiente.
3. Pegar como primer mensaje.
4. Al finalizar la sesion, Claude genera el prompt de la siguiente sesion.
5. Guardar ese prompt como el siguiente archivo y actualizar esta tabla.
