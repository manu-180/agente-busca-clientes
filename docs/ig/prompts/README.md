# Prompts de sesión

Cada sesión de Claude Code tiene un prompt acá. La modalidad:

1. Abrís una sesión nueva de Claude Code (contexto limpio)
2. Setteás el modelo que indica el archivo (`/model claude-opus-4-7` o `/model claude-sonnet-4-6`)
3. Copiás y pegás el contenido del `SESSION-XX.md` correspondiente
4. Claude trabaja la sesión y al final:
   - Actualiza `PROGRESS.md`
   - Crea el próximo `SESSION-(XX+1).md` acá
   - Te dice qué modelo y prompt usar en la próxima
